Evennia 是一款开源的MUD游戏服务器软件，它使用Python编写，基于Twisted和Django开发，使用BSD许可协议发布，对商业应用友好。
虽然MUD是一种历史比较久远的游戏方式，但Evennia自身使用了不少新技术。它结构清晰、文档丰富、开发简便、易于扩展，还自带有网页服务器和客户端，加以改动就可用于更广泛的用途，发展潜力很大。

这是一个很小的适合单人游玩的游戏，游戏情节非常简短，仅用于展示Evennia系统的特性。游戏内容源自Evennia的tutorial world。由于会不定期地更改游戏代码及数据结构，用户的游戏进度、数据可能会部分或全部丢失，敬请谅解。

该游戏使用了链接式的操作模式，除了在登录时需要输入用户名和密码外，其余时间玩家可以完全通过点击来进行游戏。当然，你依然可以通过键入命令进行游戏，但你的客户端需要能输入中文。由于使用了链接功能，某些Mud客户端可能无法支持该游戏，建议使用网页客户端。

如果想了解更多信息，欢迎访问Evennia中文站。
